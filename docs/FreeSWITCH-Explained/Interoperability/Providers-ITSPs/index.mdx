
# Providers ITSPs 

 

## About

This section provides examples of how to connect FreeSWITCH to a variety of Internet Telephone Service Providers.

If you add or change a page please observe these rules!

* If you update a provider page, remove the update note at the top of the page. This is used to carry last update dates from the old wiki and is not needed when the page is updated.
* include a generic configuration
* Do not include marketing information
* Include the country of the main office location next to the name

xx

Each provider has the country where it's main office is located in next to the name. In many cases the provider handles multiple countries.
