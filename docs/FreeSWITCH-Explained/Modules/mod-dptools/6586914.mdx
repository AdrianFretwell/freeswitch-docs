## Description

Clears out a channel variable.

## Usage

```xml
<application="unset" data="variable_name">
```

This application may be run inline from the [XML dialplan](../../Dialplan/XML-Dialplan-archive_6586601.mdx#about).

## Example

```xml
<action application="unset" data="foo"/>
```
