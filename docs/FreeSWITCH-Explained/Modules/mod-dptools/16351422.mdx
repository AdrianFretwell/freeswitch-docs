
# mod_dptools: loop_playback 

 Created by  livem Chan, last modified on 2017.11.08

## About

Playback a file to the channel looply for limted times. **But it can not set pause time in between,** like [playback](./6586776.mdx#0-about)，[playback\_sleep\_val](../../Dialplan/Channel-Variables_16352493.mdx#channel-variable-scope-example)，[file\_string](./6586646.mdx#0-about).

## Usage

```bash
loop_playback [+loops] <path>
```

## Example

Play a file twice at the same time.

```xml
<action application="loop_playback" data="+2 path.wav"/>
```
