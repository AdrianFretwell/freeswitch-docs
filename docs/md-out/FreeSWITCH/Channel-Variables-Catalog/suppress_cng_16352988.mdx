
# suppress_cng 

 Created by  Ryan Harris, last modified on 2018.02.07

## suppress_cng

boolean Sets `a=silenceSupp: off` in the SDP to disable silence suppression while making an outbound call.

**Usage**

```xml
<action application="set" data="suppress_cng=true"/>
```
