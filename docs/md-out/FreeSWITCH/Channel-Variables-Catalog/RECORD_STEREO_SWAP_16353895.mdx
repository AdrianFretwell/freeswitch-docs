## RECORD_STEREO_SWAP

boolean Allow to invert the recording channel when [RECORD\_STEREO](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#record_stereo) variable is set to `true`. So the caller is recorded to the right channel and the receiver is recorded on left channel.
