
# Provider Xpander Communications (USA) 

 Created by  Mario G, last modified on 2014.05.31

## [www.xpandercommunications.com](http://www.xpandercommunications.com)

Last update: 3 August 2012

* Inbound Calls  
   * send a destination\_number with 10 digits.  
Outbound Calls  
   * outbound caller id works with sip\_cid\_type=none, caller-id-from=true  
FreeSWITCH Gateway XML

```xml
 <include>
     <gateway name="xpander">
       <param name="username" value="register"/>
       <param name="password" value="false"/>
       <param name="realm" value="xpander"/>
       <param name="proxy" value="t2292.xpandervoip.net"/>
       <param name="expire-seconds" value="800"/>
       <param name="register" value="false"/>
       <param name="retry-seconds" value="60"/>
       <param name="context" value="public"/>
       <param name="caller-id-in-from" value="true"/>
       <param name="sip_cid_type" value="none"/>
     </gateway>
 </include>
```
