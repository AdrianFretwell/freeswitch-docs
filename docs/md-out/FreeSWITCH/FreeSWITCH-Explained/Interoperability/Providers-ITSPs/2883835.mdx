
# Provider Rapidvox (USA) 

 Created by  Mario G, last modified on 2014.05.31

## [www.rapidvox.com](http://www.rapidvox.com)

Last update: 29 July 2010

* Works outbound, outbound provider only.
* Add to conf/sip\_profiles/external/rapidvox.xml
* Replace username and password with your credentials.

```xml
<include>
     <gateway name="rapidvox">
        <param name="username" value="username"/>
        <param name="realm" value="sip.rapidvox.com"/>
        <param name="password" value="password"/>
     </gateway>
</include>
```
