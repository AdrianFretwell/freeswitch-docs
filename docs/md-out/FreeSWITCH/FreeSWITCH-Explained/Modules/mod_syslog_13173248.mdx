
# mod_syslog 

 Created by  Artur Lins Barros Oliveira on 2016.12.23

The possible ${VARNAME} substitutions in the format param are:

```xml
${time} - time in %Y-%m-%d %T format
${file}
${func}
${line}
${message}
```

There is currently no way to insert the level, nor to change the date format.
