
# mod_dptools: break 

 Created by  Belaid Areski, last modified by  John Boteler on 2016.01.18

# About

Cancel an application currently running on the channel.Dialplan execution proceeds to the next application. Optionally clears all unprocessed events (queued applications) on the channel.

# Usage

break [all]

# Example

```xml
<action application="break" data="${bridge_to}">
```

Note: Currently you can't break an endless\_playback
