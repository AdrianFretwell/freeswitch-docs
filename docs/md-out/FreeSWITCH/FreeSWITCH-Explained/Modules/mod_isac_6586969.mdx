
# mod_isac 

 Created by  Belaid Areski on 2014.11.14

# About

iSAC is a HD codec kit similar to G.722 anthony extracted iSAC codec from WebRTC project and built mod\_iSAC on top of it.

# How to build

cd to your freeswitch top directory like: cd /usr/src/freeswitch if you have allready bootstraped and configured your freeswitch, then do the following:

* edit modules.conf
* add: codecs/mod\_iSAC

save, and do make && make install
